git tag 跟有多个分支差不多
git tag => 列出所有的tag git tag -l "v3.3.*" => -l通配符是过滤tag git tag 版本号 => 新建一个tag git tag -a tagName -m "xxx" => -a 表示创建一个带备注的tag; -m 后面是备注信息 git show tagName => 可以查看tag 的详细信息,包括commit git tag -a v1.2 9fceb02 -m "my tag" 给某一个commit加tag 并备注信息 git push origin tagName => 将tag 同步到远程 git push origin --tags => 推送本地所有tag git checkout tagName => 切换到某个tag git tag -d tagName => 本地删除某个tag git push origin :refs/tags/tagName => 远程删除tag
